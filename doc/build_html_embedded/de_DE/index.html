Willkommen zur Dokumentation von SpeedCrunch!

SpeedCrunch ist ein sehr genauer wissenschaftlicher Taschenrechner. Es wurde entwickelt um per Tastatur bedient zu werden. Seine Funktionen umfassen u.a. Syntaxhervorhebung, scrollbares Display, automatische Vervollständigung von Funktionen und Variablen, sowie eine Formel- und Konstantensammlung. SpeedCrunch läuft auf Windows, OS X und Linux, und ist in einer Reihe von Sprachen erhältlich.
